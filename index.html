Programador Brian Lewis propõe solução inovadora para combater golpes no Pix

Projeto já foi enviado para grandes bancos e fintechs do Brasil

O aumento expressivo dos golpes em transações via Pix tem preocupado milhões de brasileiros. Pensando em trazer mais segurança para esse meio de pagamento instantâneo, o jovem programador web Brian Lewis, de apenas 15 anos, apresentou uma proposta de solução simples e eficaz.

O projeto consiste em disponibilizar um painel individual para cada usuário, onde é possível acompanhar as transações Pix recebidas e contestar eventuais golpes em até 5 dias úteis, garantindo a restituição do valor quando houver provas suficientes.

Envio do projeto para bancos e fintechs

Brian enviou a proposta para análise de mais de cinco importantes instituições financeiras e fintechs brasileiras, entre elas:

  • Banco Inter
  • Nubank
  • PicPay
  • Banco Mercantil
  • Caixa Econômica Federal

Além dessas, o projeto também foi encaminhado para outras entidades do setor, visando sua ampla adoção.

Detalhes da solução

  1. Painel personalizado para cada usuário: o usuário poderá acompanhar todas as transações em um painel digital.
  2. Prazo de contestação: até 5 dias úteis para informar golpes.
  3. Provas para restituição: devolução do dinheiro mediante comprovação.
  4. Segurança avançada: mecanismos para evitar fraudes e contas falsas.
